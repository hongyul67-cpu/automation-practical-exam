# 📐 자동화설비기능사 실기 · 공개도면 문제 게임

자동화설비기능사 **실기(공개도면)** 대비 문제 게임입니다. 공개도면을 여러 방식으로
연습하는 3가지 모드와 편집기를 제공합니다.

🔗 **바로가기:** https://hongyul67-cpu.github.io/automation-practical-exam/

## 주요 기능
- **3가지 문제 모드** — 도면 해독 · 3D 매칭 · 치수 찾기
- **도면 편집기** — 문제를 직접 만들고 편집
- 공개도면 이미지 기반 학습 (`images/` 폴더)

## 결과수집
학습 결과를 교사 구글시트로 제출할 수 있습니다. (공유 링크의 `?rc=` 파라미터로 활성화)

## 사용 방법
- 위 주소를 열면 바로 사용할 수 있습니다. 설치가 필요 없습니다.
